# KyvShield Plugin

This plugin teaches Claude how to integrate the KyvShield KYC SDK into any application.

## Skills included

- `/kyvshield` — Complete integration guide with examples for all platforms (Web, Flutter, Android/Kotlin, iOS/Swift, React Native) and webhook handling.

## What KyvShield does

KyvShield provides identity verification (KYC) with:
- Selfie liveness detection (anti-spoofing)
- Document capture and OCR (ID cards, passports, driver licenses)
- Face matching (selfie vs document photo)
- Multi-language support (French, English, Wolof)

## Usage

Ask Claude to integrate KyvShield into your project:

```
Help me add KYC verification to my Flutter app
```

Or invoke the skill directly:

```
/kyvshield flutter
/kyvshield android
/kyvshield swift
/kyvshield web
/kyvshield webhooks
```
